# Sanctus Caelum - Official Game Website

**Sanctus Caelum** is a MOBA (Multiplayer Online Battle Arena) concept game featuring iconic anime characters as playable champions. This repository contains the official game website currently under active development.

## 🎮 Game Overview

A competitive MOBA that brings together beloved anime universes into a single strategic experience. Champions are drawn from universes including Naruto, Bleach, Jujutsu Kaisen, Hunter x Hunter, Dragon Ball, Tokyo Ghoul, Fire Force, Black Clover, and more.

Each champion has unique abilities, resource systems, roles, and playstyles inspired by their original source material.

## 🌐 Project Phases

### Phase 1: Champions Showcase ✅ In Progress

Official platform to browse, filter, and explore all game champions.

- Location: `champions-web/`
- Features: Champion grid, filtering, suggestions, contact forms
- Status: Core functionality complete

### Phase 2: Game Home (📋 Planned)

Main landing page introducing the game, concept, and universe.

- Location: `game-home/`
- Features: Game overview, feature highlights, getting started guide

### Phase 3: World Building (📋 Planned)

Encyclopedia of game maps, modes, and creatures.

- Location: `game-worldbuilding/`
- Features: Map guides, game mode explanations, monster database

### Phase 4: Ranked System (📋 Planned)

Competitive ranking, tier system, and matchmaking mechanics.

- Location: `ranked-system/`
- Features: Tier explanations, ranking algorithm, leaderboard system
